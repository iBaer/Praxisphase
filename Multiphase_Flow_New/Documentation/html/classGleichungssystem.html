Detailed Description

Die Klasse Gleichungssystem verwaltet die Formelvektoren u,f_u und s_u. Die Formelvektoren wurden in der ursprüngliche Version als Strings in Vektoren abgespeichert, und dann mittels Funktionen der exprtk Libiary interpretiert, aber das dauerte viel zu lange
