Detailed Description

Diese Klasse bietet ein Raster aus Zellen. Zellen werden in einem 1-Dimensionalen Array abgespeichert, welches dann auch noch falls nötig 2-Dimensional oder 3-Dimensional interpretiert wird.

Zellen werden in einem 2-dimensionalen Array abgespeichert. Die erste Dimension entspricht der Position und wird dann je nach Dimension des Rasters interpretiert. Die zweite Dimension entspricht den Zellwerten in der Reihenfolge: d, p, ux, uxr, uy, ...
